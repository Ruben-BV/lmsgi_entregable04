Inicio

Esta página web tiene como objetivo ofrecer diversos recursos de los tres lenguajes fundamentales que forman la base de cualquier sitio web moderno: HTML, CSS y JavaScript.

HTML (HyperText Markup Language) es el lenguaje de marcado que estructura el contenido en la web. Gracias a él, podemos definir encabezados, párrafos, listas, imágenes, enlaces y mucho más. Es, por así decirlo, el esqueleto de toda página web.

CSS (Cascading Style Sheets): se encarga del diseño visual. Permite aplicar estilos al contenido estructurado con HTML: colores, fuentes, adaptabilidad a distintos dispositivos, etc.

JavaScript: aporta dinamismo e interactividad. Con él, una página deja de ser estática y puede responder a acciones del usuario, validar formularios o mostrar animaciones, entre otros.
